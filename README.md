
# 🚀 Kasım Uğur | Modern Portfolio v2

This project is a modern, high-performance, and SEO-focused portfolio website built with **Next.js 16**, **Tailwind CSS v4**, and **Framer Motion**.

![Portfolio Preview](https://via.placeholder.com/1200x600?text=Kasim+Ugur+Portfolio+Preview)

---

## 📈 SEO & Metadata

The project is fully optimized for search engines and social media platforms:

- **Dynamic Title & Description:** Page-based metadata for better visibility
- **OpenGraph & Twitter Cards:** Professional sharing previews (`summary_large_image`)
- **Metadata Base:** Standard-compliant dynamic URL structure

---

## ✨ Features

- 🌗 **Dynamic Theme System:** Smooth dark/light mode with `next-themes`
- 🛠 **Semantic Design System:** Centralized styling using Tailwind v4 + CSS variables
- 🚀 **Modern Engineering:** SSR, URL-based state management, and Error Boundaries
- ✨ **Smooth Animations:** Enhanced micro-interactions with Framer Motion
- 📱 **Fully Responsive:** Mobile-first design across all devices

## 📂 Project Structure

```text
├── app/                # Pages, layouts, and SEO metadata
│   └── globals.css     # Tailwind v4 config and theme variables
├── components/         # Reusable UI components
│   ├── sections/       # Page sections (Hero, Skills, Projects)
│   └── ui/             # Base UI elements (Button, Badge, etc.)
└── public/             # Static assets (images, logos)
```

## 🚀 Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/kasimugur/portfoliov1.git
```

2. **Install dependencies:**

```bash
npm install
```

3. **Run the development server:**

```bash
npm run dev
```
